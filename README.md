Multiple implementations of Shell, Awk, Sed
===========================================

Adjusted to compile cleanly on Debian GNU/Linux.

To build, run:

    git clone https://github.com/agordon/sas-implementations
    cd sas-implementations
    make

Upon successful build, the directory ./bin/ will contain:

    bash-3.2.57
    bash-4.3.30
    gawk-3.1.8
    gawk-4.1.3
    gsed-3.02
    gsed-4.0.6
    gsed-4.2.2
    busybox-ash
    busybox-awk
    busybox-httpd
    busybox-sas
    busybox-sed
    freebsd-10-awk
    freebsd-10-sed
    heirloom-nawk
    heirloom-nawk_su3
    heirloom-nawk_sus
    heirloom-oawk
    heirloom-sed
    heirloom-sed_s42
    heirloom-sed_su3
    heirloom-sed_sus
    heirloom-sh
    netbsd-7.0-awk
    netbsd-7.0-sed
    openbsd-5.9-awk
    openbsd-5.9-sed
